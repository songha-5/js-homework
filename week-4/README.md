
- **기술적 적합성 :** `querySelectorAll`과 `forEach`를 활용한 반복 처리 능력

- **데이터 활용 :** HTML의 `data-plan`, `data-price` 속성을 JS에서 올바르게 읽어왔는지
- **가독성 및 문서화 :** 변수명(예: `cards`, `selectedCard`), 코드 품질 및 README 작성 충실도
- **접근성 준수 :** 스크린 리더 인식 및 키보드 운용 여부
